# Piedra Papel o tijeras
En este proyecto se realizo un juego de piedra papel o tijeras progamado en python donde el usuario elije una de las tres opciones y compite contra la computadora.
# Funcionamiento
- El usuario elije una de las tres opciones.
- Se compara los resultados con el elegido por la computadora y se elije un ganador
- Puede dar el caso en cual el juego termine en empate cuando ambas opciones son iguales
# Estrucuturas Utilizadas
- Variables
- Condicionales
- Bucles (while)
- Entrada y salida de datos proporcinados por el usuario
